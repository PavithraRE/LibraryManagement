Library Management System

This is a Java-based web application developed using Servlets,and JDBC.
It allows users to add and search books using ISBN.
The project follows a layered architecture with Bean, DAO, Service, and Servlet components.
Oracle Database is used for storing book and author details.
The application runs on Apache Tomcat server.
